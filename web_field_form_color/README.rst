==========================
Web Field Form Color Rules
==========================

This module allows to define color rules on form views such as in list views.

Usage
=====

To use this module, you need to add a 'colors' attribute on a float field in
a form view.

.. image:: https://odoo-community.org/website/image/ir.attachment/5784_f2813bd/datas
   :alt: Try me on Runbot
   :target: https://runbot.odoo-community.org/runbot/162/8.0

Known issues / Roadmap
======================

For now, only the float fields are supported. The use case is to have amounts
in different color (red for example) when an amount is between a given range.
The module could be improved later if other types of fields are required.
